# 클났으505(Clnass505)

<br>

## 프로젝트 소개
- clnass505(클났으SOS)는 class101을 모티브로 하여 일상 생활 속의 도움이 필요한 상황들에 대한 강의를 제공하는 사이트로, 강의의 수강과 생성에 대한 로직을 로그인과 비로그인으로 나누어 구현함에 포커스를 맞추고 클론 코딩을 진행하였습니다.
- 짧은 기간 동안 해당 사이트의 기획/디자인만을 참고하여 프로젝트를 진행했습니다.
<br>
